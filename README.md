# Korbáčky od babičky Marušky | Ručně vyráběné dárky s láskou

Vytvoř jednoduchý, přehledný a milý web pro babičku, která ručně vyrábí korábky. Styl má být útulný, domácí a důvěryhodný, s jemnými barvami a velkými dobře čitelnými písmy.

Stránka má obsahovat:
- úvodní sekci s názvem značky a krátkým popisem
- galerii fotek korábků
- sekci s cenami
- sekci o výrobě a použitých materiálech
- kontakt
- možnost objednávky přes e-mail nebo telefon

Design:
- jednoduchý layout
- mobilně responzivní
- příjemné barvy: krémová, béžová, světle hnědá
- velké tlačítko na objednání
- žádné složité animace

Texty mají být v češtině a působit osobně a srdečně.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
